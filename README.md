# Daily Tailwind CSS Practice - Card Component

## Project Overview
This project is part of my daily Tailwind CSS practice, where I recreate UI components to improve my skills. Today's component is a card design inspired by @Aj_Dzine's work from the Figma Community.

## Original Design Credit
- **Designer:** @Aj_Dzine
- **Source:** [Figma Community](https://www.figma.com/@Aj_Dzine)

## Learning Goals
- Practice Tailwind CSS utility classes
- Improve component layout skills
- Master responsive design principles
- Understand flexbox and grid systems in Tailwind

## Technologies Used
- HTML
- Tailwind CSS

## How to Run
1. Clone this repository
2. Make sure you have Tailwind CSS installed
3. Open the HTML file in your browser

## Component Features
- Responsive design
- Clean and modern layout
- Proper spacing using Tailwind's utility classes
- Optimized for different screen sizes
